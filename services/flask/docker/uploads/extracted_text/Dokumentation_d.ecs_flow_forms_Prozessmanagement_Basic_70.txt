11.5. Workflow-Variablen

Fur die Steuerung des Prozesses innerhalb der Workflow-Verarbeitung werden im d.ecs flow JPL-
Variablen bendtigt, welche entweder in einer Aktion mittels JPL initialisiert werden mussen oder Uber
entsprechende Konfigurationen im Formular erzeugt und geschrieben werden.

Innerhalb eines JPL-Skriptes:

 

// STEP 0: Initialisierung der Variablen
formInstanceld "XX-" ## doku_id ## "-" ## LUUID
vars wSuperior dok_dat_feld[29]

vars wApplicaant dok_dat_feld[18]

vars wStepID 1

vars wState ue

// STEP 1: Glolbalisieren der Variablen
call save_vars( "formInstancelId" )

call save_vars( "wSuperior" )
call save_vars( "wApplicant" )
call save_vars( "wStepID" )
call save_vars( "wState" )
