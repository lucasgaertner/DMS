Modals

Forms Lightbox Dialogs Documents PopUp Modals

A lightweight multi-purpose JavaScript popup that is customizable and responsive. It can be used to display e. g. terms and conditions as a part of a sign-up process, as a wizard-like form or just to show extra information in front of your content.

Basic modal

Individual usage

The modal is a jQuery plugin, so if you want to control the modal using jQuery, then you need to call the .modal() function on the modal‚Äôs selector.

$('#basicModal').modal(options);

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-backdrop="".

Name type default description
backdrop boolean or the string 'static' true Includes a modal-backdrop element. Alternatively, specify static for a backdrop which doesn't close the modal on click.
keyboard boolean true Closes the modal when escape key is pressed
show boolean true Shows the modal when initialized.

Events

You can further customize the normal behaviour of the modals by using various events that are triggered while opening and closing the modal. These events have to be bound using jQuery‚Äôs .on() method.

Event Type Description
show.bs.modal This event fires immediately when the show instance method is called. If caused by a click, the clicked element is available as the relatedTarget property of the event.
shown.bs.modal This event is fired when the modal has been made visible to the user (will wait for CSS transitions to complete). If caused by a click, the clicked element is available as the relatedTarget property of the event.
hide.bs.modal This event is fired immediately when the hide instance method has been called.
hidden.bs.modal This event is fired when the modal has finished being hidden from the user (will wait for CSS transitions to complete).
loaded.bs.modal This event is fired when the modal has loaded content using the remote option. 
$('#myModal').on('hidden.bs.modal', function (e) {
   do something...
 })

Tabs

Enable tabbable tabs via JavaScript (each tab needs to be activated individually):

$('#myTab a').click(function (e) {
e.preventDefault()
$(this).tab('show')
})

Individual usage

You can activate individual tabs in several ways:

$('#myTab a[href="#profile"]').tab('show') // Select tab by name
$('#myTab a:first').tab('show') // Select first tab
$('#myTab a:last').tab('show') // Select last tab
$('#myTab li:eq(2) a').tab('show') // Select third tab (0-indexed)

Tooltip

Tooltips don't rely on images, we use CSS3 for animations, and data-attributes for local title storage.

Example

<button type="button" class="btn btn-default tip" data-toggle="tooltip" data-placement="left" title="Tooltip: left">left</button>
<button type="button" class="btn btn-default tip" data-toggle="tooltip" data-placement="top" title="Tooltip: top">top</button>
<button type="button" class="btn btn-default tip" data-toggle="tooltip" data-placement="bottom" title="Tooltip: bottom">bottom</button>
<button type="button" class="btn btn-default tip" data-toggle="tooltip" data-placement="right" title="Tooltip: right">right</button>
<script>window.onload = function() { jQuery(document).ready(function() { jQuery('.tip').tooltip(); }); }</script>
